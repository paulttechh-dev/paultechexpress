# 🚀 PAULTECH EXPRESS

**PAULTECH EXPRESS** é uma plataforma mobile-first moderna para busca e entrega de celulares para assistência técnica. O sistema conecta clientes, motoboys e administradores em um fluxo ágil e seguro de reparo de aparelhos.

---

## ✨ Funcionalidades

### 👤 Portal do Cliente
- **Cadastro Simples**: Crie sua conta em segundos.
- **Solicitação de Coleta**: Informe o endereço e o problema do aparelho.
- **Acompanhamento Real-time**: Veja o status do seu pedido mudar de "Aguardando" para "Entregue".

### 🛵 Portal do Motoboy
- **Gestão de Corridas**: Visualize pedidos disponíveis e aceite novas entregas.
- **Atualização de Status**: Informe quando o aparelho foi coletado, levado à manutenção ou entregue.
- **Histórico**: Acompanhe suas entregas realizadas.

### 🛡️ Painel Administrativo
- **Gestão de Equipe**: Cadastre e gerencie motoboys.
- **Monitoramento**: Visualize todos os pedidos da plataforma.
- **Métricas**: Dashboard com estatísticas de pedidos e produtividade.

---

## 🛠️ Tecnologias

- **Frontend**: React 19 + Vite + Tailwind CSS 4.
- **Backend**: Node.js + Express (Full-stack).
- **Banco de Dados & Auth**: Firebase (Firestore + Authentication).
- **Animações**: Motion (Framer Motion).
- **Ícones**: Lucide React.

## ⚙️ Configuração Local

### 1. Clonar o Repositório
```bash
git clone https://github.com/seu-usuario/paultech-express.git
cd paultech-express
```

### 2. Instalar Dependências
```bash
npm install
```

### 3. Configurar Firebase
1. Crie um projeto no [Firebase Console](https://console.firebase.google.com/).
2. Ative o **Firestore Database** e o **Authentication** (Google Login ou Email/Senha).
3. Copie as configurações do seu app web e crie um arquivo `firebase-applet-config.json` na raiz do projeto seguindo o modelo de `firebase-applet-config.example.json`.

### 4. Rodar o Projeto
```bash
npm run dev
```
